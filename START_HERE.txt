🚀 QUICK START (5 MINUTES)
═════════════════════════════════════════════════════════════════════════

Step 1: Install Dependencies
────────────────────────────────
cd backend
npm install

Step 2: Configure Database (REQUIRED)
──────────────────────────────────────
Edit: backend/.env
Add MongoDB URI:
  MONGODB_URI=mongodb://localhost:27017/lost-found

Step 3: Start Backend
─────────────────────
cd backend
npm start
✓ Should see: "Server is running on http://localhost:5000"

Step 4: Start Frontend (new terminal)
──────────────────────────────────────
cd frontend
python -m http.server 8000

Step 5: Open Browser
─────────────────────
Go to: http://localhost:8000
✓ You should see the home page!

📞 TROUBLESHOOTING QUICK REFERENCE
═════════════════════════════════════════════════════════════════════════

Issue: MongoDB Connection Error
→ Fix: Check MongoDB is running, verify connection string in .env

Issue: Port 5000 already in use
→ Fix: Change PORT in .env or kill the process using port 5000

Issue: Frontend shows blank page
→ Fix: Use python -m http.server 8000, not file:// protocol

Issue: Can't login/register
→ Fix: Ensure backend is running on http://localhost:5000

See detailed troubleshooting: README.md or WINDOWS_SETUP.md
