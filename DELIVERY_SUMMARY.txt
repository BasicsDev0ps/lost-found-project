## 🚀 QUICK START (3 STEPS)

1. SETUP BACKEND:

   npm install
   (edit .env with your MongoDB URI)
   npm start

2. SETUP FRONTEND:
   cd frontend
   python -m http.server 8000

3. OPEN BROWSER:
   http://localhost:8000

## 🔗 API ENDPOINTS (10+ ENDPOINTS)

  POST /api/auth/login

Users:
  GET /api/users
  GET /api/users/:id
  GET /api/users/profile/me
  PUT /api/users/:id
  DELETE /api/users/:id

Items:
  GET /api/items (with filters)
  GET /api/items/:id
  POST /api/items (create)
  PUT /api/items/:id (update)
  DELETE /api/items/:id
  POST /api/items/:id/claim
  POST /api/items/:id/verify
  POST /api/items/:id/resolve
  GET /api/items/search/query?q=...
