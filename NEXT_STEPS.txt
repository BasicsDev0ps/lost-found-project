❓ Troubleshooting

Problem: "I see blank page"
  → Hard refresh: Ctrl+Shift+R
  → Check backend is running
  → Check http://localhost:8000 (not file://)

Problem: "Still no items showing"
  → Run: npm run seed (in backend folder, new terminal)
  → Wait for success message
  → Refresh browser

Problem: "Can't upload photo"
  → Click "Choose File", select JPG/PNG/GIF
  → File should appear in form
  → Submit form - image will be saved

Problem: "Login doesn't work"
  → Check email spelling: madhuravani@campus.edu
  → Check password: password123
