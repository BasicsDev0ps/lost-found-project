## 🔐 IP Whitelist Fix (IMPORTANT!)

**Current Issue**: Server can't connect because your IP is not whitelisted

### Steps to Fix:

1. **Visit MongoDB Atlas**:
   ```
   https://cloud.mongodb.com/
   Log in with your account
   ```

2. **Navigate to Security → Network Access**:
   ```
   Click your cluster → Security tab → Network Access
   ```

3. **Add Your IP**:
   ```
   Option A: Add your current public IP (recommended)
   Option B: Allow 0.0.0.0/0 (all IPs - for dev only)
   ```

4. **Restart Server**:
   ```powershell
   cd backend
   Get-Process node | Stop-Process -Force
   node server.js
   ```
